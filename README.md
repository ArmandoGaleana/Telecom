# Telecom
código para pronosticar la tasa de cancelación de clientes.

## En resumen.
## Este código fue creado para pronosticar la tasa de cancelación de clientes que solicitó el operador de "telecomunicaciones Interconnect". Para poder realizar dicha tarea se emplearon dos modelos de predicción para compararlos y poder ver que modelo otorga mejores resultados.
## Se evaluaron los modelos Gradient Boosting y Logistic Regression, de los cuales se observó que el que mejor resultó fue Gradient Boosting arrojando una métrica AUC-ROC y una Accuracy de 0.89 y 0.85 respectivamente.
